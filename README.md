# Bot RAG de Ventas con Mistral

Bot que responde preguntas sobre datos de ventas usando RAG + Mistral API.

## Stack
- Mistral API
- LangChain
- FAISS
- Sentence Transformers

## Cómo usar
1. Abrir el notebook en Google Colab
2. Configurar la API Key de Mistral
3. Subir el CSV y ejecutar todas las celdas
